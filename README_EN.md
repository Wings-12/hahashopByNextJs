## Overview
This is a website for a flower and clothing store that my mother plans to manage.

## Requirements
- Node.js 18.x
- npm 9.x
- Next.js 13.x

# Setup
- After cloning the repository, install dependencies with the following command:
```
nmp install
```
- Start the Next.js development server:
```
npm run dev
```
