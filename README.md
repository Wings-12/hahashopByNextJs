## 概要
母が経営予定の花と服屋のウェブサイトです。

## 動作環境
- Node.js 18.x
- npm 9.x
- Next.js 13.x
- Supabase

## 環境構築
- リポジトリをクローン後、以下のコマンドで依存関係をインストールしてください。
```
nmp install
```
- Next.js の開発サーバーを起動します。
```
npm run dev
```
